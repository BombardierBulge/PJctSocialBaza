# PJctSocialBaza Backend

Backend for PJctSocialBaza using Node.js, TypeScript, and PostgreSQL.

## Setup

1. Install dependencies: `npm install`
2. Set up PostgreSQL database and update .env file with your credentials.
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
5. Start production server: `npm start`

## Future Frontend

This backend is prepared for integration with a React frontend.
